# Halo Safe — Deck Treatment

Editorial / brand-deck rendering of the Halo Safe website.

- Single static HTML + tokens CSS + assets
- Foundation tokens in `colors_and_type.css` (sourced from the iOS app design system)
- Inter Tight (display) + JetBrains Mono (chrome labels) via Google Fonts
- Hero orb is `assets/orb.png`
- Wordmark in three variants: `logo-white.svg`, `logo-black.svg`, `logo-gradient.svg`

Noindex by default per workspace standing rule.
